Injection de dépendances - Minification

Après minification ce code :

angular.module('myModule').controller('MyCtrl', function ($scope, $http) {});

deviendra :

angular.module('myModule').controller('MyCtrl', function (a, b) {});

AngularJS recherchera des providers aProvider et bProvider qui n'existeront certainnement pas.

Pour palier à ce problème, on peut utiliser la forme :

 angular.module('myModule')
    .controller('MyCtrl', ['$scope', '$http', function ($scope, $http) {}]);
