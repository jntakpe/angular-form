## Héritage des scopes

* Lors de démarrage de l'application un objet $rootScope est créé, il sera le parent de tous les scopes.
* Héritage prototypal des scopes.

Exemple

    git checkout scope-inherit

Autre exemple

    git checkout scope-inherit-weird
